# 项目介绍
本项目是基于STM32F103C8T6的FreeRTOS移植示例工程，内包含OLED是0.96寸四线I2C的

移植流程：https://yesord.github.io/post/FreeRTOS%E7%9A%84%E7%A7%BB%E6%A4%8D
FreeRTOS资料详细可参考 https://www.freertos.org/a00110.html
